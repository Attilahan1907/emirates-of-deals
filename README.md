# Emirates of Deals

Echtzeit-Preisvergleich für Kleinanzeigen.de & eBay.

## Features

- Preisvergleich auf Kleinanzeigen.de und eBay
- Deal-Score Algorithmus (GPU/CPU/RAM/Smartphone Benchmarks)
- Preis-Alarme via Telegram
- Watchlist / Favoriten
- Suchalarme für neue Angebote

## Tech Stack

- Backend: Python, Flask, BeautifulSoup4
- Frontend: React 18, Vite, Tailwind CSS

## Starten

```bash
start-all.bat
```

Backend: http://localhost:5000 · Frontend: http://localhost:5173
